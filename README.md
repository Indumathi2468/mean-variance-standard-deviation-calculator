# Mean-Variance-Standard-Deviation Calculator

This project calculates the **mean, variance, standard deviation, maximum, minimum, and sum** of a 3×3 matrix using Python and NumPy.

The logic is implemented in a reusable function, with a separate main file for execution and a test file for validation.

---

## Project Description

The program accepts a list of 9 numbers, reshapes it into a 3×3 matrix, and performs statistical calculations:
- Column-wise
- Row-wise
- Overall

Results are returned as a dictionary.

---

## Technologies Used
- Python 3
- NumPy
- unittest (Python standard library)
